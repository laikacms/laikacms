date: 2026-05-31T10:00:00.000Z

This starter uses the **legacy Pages Router** (`pages/` directory) — what most existing Next.js
projects still ship. Server data goes through `getServerSideProps` instead of async server
components.

LaikaCMS doesn't care which routing system you use; the embedded preset works the same way. This
variant exists because migrating from Pages Router to App Router is a separate effort, and folks
still on Pages should not have to wait.
