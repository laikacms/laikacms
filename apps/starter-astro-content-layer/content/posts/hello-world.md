Welcome to a static blog powered by Astro's Content Layer and LaikaCMS!

Blog posts are prerendered at build time using LaikaCMS's `listRecords` API directly — no HTTP
round-trip during the build. The admin UI runs as a live SSR endpoint so you can edit content and
then rebuild to publish.
