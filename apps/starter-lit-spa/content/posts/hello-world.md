This starter is **framework-less** on the client — just web standards, with Lit as a 5KB authoring
library on top of the native Custom Elements + Shadow DOM APIs. No React, no Vue, no Svelte. The
`<post-list>` and `<post-detail>` custom elements own their data fetching and rendering.

It's also the **first starter to use `decapAdminHtml()`** — the new
`@laikacms/decap-integrations/embedded` helper that replaces the ~50-line admin HTML file every
previous starter shipped.
