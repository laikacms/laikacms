Welcome to a blog backed by WebDAV storage!

Content lives on any RFC 4918 WebDAV server — Nextcloud, ownCloud, Apache mod\_dav, nginx-dav, or
the embedded dev server (running locally on port 4918). Swap `WEBDAV_URL` to point at your
production WebDAV server without changing any application code.
