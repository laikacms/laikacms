This Jamstack starter renders posts at **build time** with Eleventy, reading the same
`content/posts/` directory that LaikaCMS uses as its storage root.

When you edit this post in the Decap admin (`pnpm admin:dev` on port 3001), the file changes on disk
and Eleventy's `--serve` mode rebuilds the affected pages. No client-side JavaScript required for
the public site.
